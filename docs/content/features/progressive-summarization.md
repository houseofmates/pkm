# progressive summarization

progressive summarization is a technique where you refine captured material in layers over time, rather than processing everything up front.

## the layers

1. **capture** — save the raw material
2. **bold** — highlight the key passages
3. **highlight** — narrow down to the essential points
4. **summary** — write a brief summary in your own words
5. **remix** — use it in your own work
