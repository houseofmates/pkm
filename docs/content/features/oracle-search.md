# oracle search

oracle is pkm's ai-powered search interface. ask it a question in natural language and it searches your entire knowledge base using [[semantic-search]].

## what makes it different

- understands intent, not just keywords
- can synthesize answers from multiple [[notes]]
- powered by [[wilson-rag]] for retrieval-augmented generation
- part of the [[ai-features]] suite

oracle treats your pkm as a [[second-brain]] you can have a conversation with.
