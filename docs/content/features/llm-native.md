# llm-native

llm-native means ai isn't an add-on — it's a core design principle. the system is built from the ground up assuming an llm is available.

## what this looks like

- [[semantic-search]] is the default search, not keyword search
- [[ai-co-pilot]] is always available, not hidden in a menu
- content is structured for retrieval by [[wilson-rag]]
- [[tags]] and [[connections]] can be suggested automatically

this is different from tools that bolt ai onto an existing design. see also: [[ai-features]], [[oracle-search]]
