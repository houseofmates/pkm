# semantic search

semantic search finds results based on meaning, not just keywords. when you search for "feeling overwhelmed," it also surfaces notes about stress, burnout, and coping strategies.

## how it works

- content is embedded into vector space using an llm
- search queries are matched by semantic similarity
- results rank by relevance, not just text matches

this powers the [[oracle-search]] feature and is part of the broader [[ai-features]] system. see also: [[search-system]]
