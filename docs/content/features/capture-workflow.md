# capture workflow

the capture workflow is the path content takes from the outside world into your knowledge system.

## the flow

1. **capture** — grab it with the [[capture-system]] or [[extension]]
2. **triage** — review your inbox, decide what's worth keeping
3. **process** — add [[tags]], [[links]], and context
4. **integrate** — connect it to existing [[notes]] and [[databases]]

this aligns with [[progressive-summarization]] — you refine material over time instead of all at once. see also: [[daily-workflow]]
