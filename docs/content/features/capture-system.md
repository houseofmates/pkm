# capture system

the capture system is a browser [[extension]] that lets you clip web content directly into your pkm.

## how it works

- click the extension icon or use a keyboard shortcut
- select what to capture: full page, selection, or quote
- content lands in your inbox as a new [[note]]
- from there, [[tagging|tag it]], [[links|link it]], or file it into a [[database]]

the goal is zero-friction capture so nothing slips through the cracks. see also: [[capture-workflow]]
