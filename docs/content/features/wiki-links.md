# wiki links

wiki links are the `[[double bracket]]` syntax for connecting pages. type `[[` and start typing to link to any page in your knowledge system.

## how they work

- `[[page name]]` creates a link to that page
- `[[page name|display text]]` lets you customize what's shown
- backlinks are automatic — linked pages know who links to them

wiki links are the primary way to build [[connections]] and support [[networked-thought]]. see also: [[links]]
