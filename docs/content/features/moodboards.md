# moodboards

moodboards are visual [[canvas-workflows|canvas]] layouts for collecting inspiration. drop in images, colors, quotes, and [[embeds]] to create a collage of vibes.

## use cases

- design direction for a project
- aesthetic references for worldbuilding
- visual journaling and emotional check-ins
- gathering ui/ux inspiration

moodboards live on the [[infinite-canvas]] and support [[block-editing]]. see also: [[visual-thinking]]
