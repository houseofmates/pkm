# obsidian sync

pkm supports importing from and syncing with obsidian vaults. your existing markdown files and [[wiki-links]] carry over.

## compatibility

- markdown files import directly
- `[[wiki links]]` are recognized natively
- frontmatter/yaml metadata maps to [[fields]]
- folder structure can convert to [[tags]] or [[collections]]

this isn't about replacing obsidian — it's about making migration painless if you choose to. see also: [[notes]], [[links]]
