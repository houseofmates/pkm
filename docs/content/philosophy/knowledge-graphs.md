# knowledge graphs

ideas don't live alone. they connect.

pkm auto-generates graphs from [[connections]].

## how it works

* [[notes]] link via [[wiki links]] or [[embeds]]
* [[tagging]] creates hubs
* [[database views]] visualize relations

## example graph

```
research note → [[tags:ai]] → ai [[database]]
                      ↓
[[canvas workflows]] ← embeds
```
