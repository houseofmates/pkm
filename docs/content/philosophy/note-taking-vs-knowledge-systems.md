# note-taking vs knowledge systems

notes capture. systems think.

## note-taking (obsidian, evernote)

* linear capture
* search-based retrieval
* manual organization

good for: daily journal, meeting notes.

limits: no computation, no structure.

## knowledge systems (pkm, notion)

* [[databases]] compute views
* [[knowledge graphs]] emergent insight
* [[automation]] evolves content

good for: research, projects, life systems.

## pkm sweet spot

[[notes]] + [[nocobase collections]] + [[canvas system]].

write freely. structure optionally. think spatially.
