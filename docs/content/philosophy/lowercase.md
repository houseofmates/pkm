# lowercase

all user-facing text in pkm is lowercase. this is a deliberate design choice, not a bug.

## why lowercase

- it creates a calm, low-pressure visual tone
- it reduces cognitive friction — no decisions about capitalization
- it signals that this is a personal, informal space
- it aligns with the neurodivergent-friendly design philosophy

any capitalization in ui chrome is considered a bug. user data (your notes, your names) is displayed as-is.
