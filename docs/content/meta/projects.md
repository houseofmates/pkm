# projects

projects in pkm are a way to group related [[notes]], [[databases]], and [[canvas-workflows|canvases]] around a goal.

## how projects work

- a project is a [[database]] record with its own fields (status, deadline, priority)
- linked [[notes]] and resources connect via [[relations]]
- track progress with [[kanban-view]] or [[table-view]]
- archive completed projects without losing context

see also: [[organizing-information]], [[creating-dashboards]]
