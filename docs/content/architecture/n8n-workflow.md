# n8n workflow

an n8n workflow is a visual automation pipeline made of connected nodes. each node performs an action — fetch data, transform it, send it somewhere.

## common pkm workflows

- **web capture** — [[extension]] → process → store in [[databases]]
- **ai enrichment** — new note → [[semantic-search]] embedding → tag suggestions
- **daily digest** — aggregate recent [[notes]] → send summary
- **backup** — export collections → push to git

workflows are managed through the n8n ui. see also: [[n8n]], [[automation]]
