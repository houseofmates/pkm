# tag databases

tag databases are [[collections]] where each record represents a tag with its own metadata — color, description, parent, usage count.

## why tag databases

- tags become first-class objects you can query and manage
- support [[nested-tags]] with parent-child [[relations]]
- enable tag-based [[database-views]] and filters
- richer than simple [[selects]] for complex taxonomies
