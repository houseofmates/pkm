# selects

selects are dropdown [[fields]] in [[nocobase-collections|nocobase]] that let you pick from a predefined list of options.

## types

- **single select** — pick one option (e.g. status: todo / in progress / done)
- **multi select** — pick multiple options (e.g. categories)

selects are great for lightweight categorization. for more structured taxonomy, use [[nested-tags]] or [[tag-databases]].
