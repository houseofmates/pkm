# links

links are the connective tissue of your knowledge system. every time you create a `[[wiki link]]`, you're building a path between two ideas.

links in pkm work like [[connections]] — they're bidirectional by default. if page a links to page b, page b knows about page a.

## why links matter

- they replace rigid folder hierarchies with organic structure
- they let you discover relationships you didn't plan for
- they support [[networked-thought]] and [[knowledge-graphs]]
