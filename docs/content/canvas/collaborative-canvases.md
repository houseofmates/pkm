# collaborative canvases

collaborative canvases let multiple people work on the same [[infinite-canvas]] in real time.

## how it works

- share a canvas with a link or invite
- cursors and edits sync live
- each collaborator can add [[blocks]], [[notes]], and [[embeds]]
- changes are tracked with history

this turns the [[canvas-system]] into a shared thinking space. see also: [[visual-thinking]]
