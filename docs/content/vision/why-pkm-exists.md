# why pkm exists

pkm was built because existing tools felt incomplete.

notion: great ideas, bad self-hosting. obsidian: great links, no databases. tldraw: great canvas, no persistence.

pkm is the hybrid:

* [[self hosted]] forever yours
* [[nocobase collections|nocobase]] + [[canvas system]]
* [[lowercase]] calm interface
* [[llm native]] - ai understands your structure

## the problem it solves

information fragmentation. ideas trapped in apps. no unified thinking environment.

pkm creates [[knowledge graphs]] across notes, databases, canvases.

## your invitation

build something that lasts. start small. [[notes]] → [[tagging]] → [[connections]] → [[second brain]].
